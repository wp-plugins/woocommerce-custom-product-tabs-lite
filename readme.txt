This plugin extends the WooCommerce e-commerce plugin by allowing a custom product tab to be created with arbitrary content.

== Description ==

This plugin extends the [WooCommerce](http://www.woothemes.com/woomember/go?r=130284&redirect=www.woothemes.com/woocommerce/) e-commerce plugin by allowing a custom product tab to be added to product view pages with arbitrary content.

The custom tab, if defined, will appear in between the 'additional information' and 'reviews' tabs.

= Plugin/Theme Support =
*This plugin should be compatible with any WooCommerce theme and has been tested with the following*

* Theme: ["Wootique" (free, by WooThemes)](http://www.woothemes.com/woomember/go?r=130284&redirect=www.woothemes.com/2011/09/wootique/)
* Theme: ["Woostore"](http://www.woothemes.com/woomember/go?r=130284&redirect=www.woothemes.com/2011/09/woostore/) (thanks to Dusan Belescak)

== Installation ==

1. Upload the entire 'woocommerce-product-tabs-lite' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit a product, then click on 'Custom Tab' within the 'Product Data' panel
